This is Disaster classification model, for 12 multiclass classification. The dataset link is provided in the dataset.txt.
-----------------------------------------------------
You can download and store the data into your local machine. For the execution of the code, you need to change the root directory location the code with your dataset location. 

Here in this 80-20 split has been done for the training, you can change the split if you want, all you need to do is change the markdown row to code and change the input there.
-----------------------------------------------------
Others things will work fine, in case of any issues you can add comments.
